# 📊 Make vs Buy – Supplier Cost Optimization Dashboard  

## 🚀 Project Overview  

This Power BI project analyzes supplier cost structures and supports strategic **Make vs Buy decisions** using scenario-based volume planning.

The dashboard helps:
- Compare suppliers based on unit cost and non-recurring expenses  
- Simulate different production volumes  
- Analyze buy scenario full cost  
- Evaluate capital investment requirements for in-house production  
- Identify cost avoidance opportunities

## 📈 Key Insights  

✔ Volume significantly impacts supplier ranking  
✔ High non-recurring costs influence short-term decisions  
✔ Capital investment changes long-term break-even point  
✔ Some parts are cost-effective to manufacture internally at high volumes
